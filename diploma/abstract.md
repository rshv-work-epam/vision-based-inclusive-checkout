# Анотація

Мета роботи — розробити систему візуального розпізнавання товарів для інклюзивної каси самообслуговування, що забезпечує доступність для людей з різними потребами. Запропоновано мікросервісну архітектуру з чотирма сервісами, засоби спостережуваності на базі OpenTelemetry та хмарне розгортання в Azure.
